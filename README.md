# Night Market for Meaning

A daily static art website for **later Wittgenstein's language games**, staged as
a nocturnal night-market and transit board where ordinary words change their
meaning by use.

Live site:
https://uoghluvm.github.io/2026-04-29-idea-language-games-night-market/

## Concept

The site treats language games as a public, lived practice rather than a cold
diagram. Words such as `light`, `right`, `bank`, `proof`, and `sorry` move through
contexts like a canteen, metro gate, family table, courtroom, clinic, and stage
door. Each pairing produces a different utterance, social pressure, and rule
feel.

Dominant aesthetic direction: urban nocturne with public signage, smoke, red
lamp pressure, amber ticket light, and hand-built procedural signal fields.

Secondary accents: cabaret red and practical workshop green, used only as
context signals.

## Tech Notes

- Static HTML, CSS, and vanilla JavaScript.
- Full-window Canvas animation with procedural nodes, route lines, pointer
  response, and context-colored pulses.
- No build dependency, framework, external media, web fonts, package CDN, or
  runtime network request.
- Responsive layout with reduced-motion handling.

## Run

Open `index.html` directly in a browser, or run the included local server:

```bash
npm start
```

Then visit `http://127.0.0.1:4173/`.

## Checks

```bash
npm run check
```

The check parses the JavaScript files and verifies that the static page keeps
required local assets, project theme copy, and no external HTTP references.
